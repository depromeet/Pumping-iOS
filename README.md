# Pumping
Tuist, TCA, SwiftUI Best Practices

## Micro Feature Architecture
대규모 서비스를 위한 마이크로 Feature Architecture를 구성하였습니다. 

### 4 Layer Module
계층 구조는 다음과 같습니다.
- Feature: 기능
- Domain: 모델
- Core: 템플릿
- Shared: 공용 모듈

### Target Type
타겟 타입은 다음과 같습니다.
- Example: 샘플 앱
- Implement: 구현부
- Tests: 테스트
- Testing: 테스트를 위한 목업
- Interface: 인터페이스
